# Product Overview

madoguchi-aiは、不動産業界向けの窓口営業支援AIアシスタントシステムです。AIアバターが顧客対応を行い、24時間365日の物件検索・提案を実現します。

## Core Capabilities

1. **AI会話エンジン** - Gemini 2.5 Flashによる自然な物件提案・条件ヒアリング
2. **音声対応システム** - Whisper（音声認識）+ Google Cloud TTS（音声合成）で高齢者にも優しいインターフェース
3. **物件取り込み** - Chrome拡張機能でアットホーム等からワンクリック取り込み、Geminiで構造化
4. **マルチテナントSaaS** - 複数の不動産会社（オペレーター）を管理者が一元管理
5. **営業メール配信** - 新着物件通知、フォローアップ、Resend連携による配信停止管理

## Target Use Cases

### 顧客向け
- 24時間いつでも音声・テキストで物件検索
- AIアバターとの対話による条件ヒアリング
- 内見予約の申し込み

### オペレーター（不動産会社）向け
- Chrome拡張でワンクリック物件登録
- 自社顧客への新着物件メール配信
- 顧客対応履歴の確認

### 管理者（gyam様）向け
- オペレーター企業の登録・サブスクリプション管理
- システム全体の統計・監査ログ確認

## Value Proposition

- **24時間対応**: AIが営業時間外も顧客対応、機会損失を削減
- **高齢者対応**: 音声入力で誰でも使いやすいUI
- **業務効率化**: 物件登録の自動化で手入力不要
- **SaaS展開**: 他社への横展開でビジネス拡大可能

## User Roles

```
Admin（gyam様）
  └─ Operator（不動産会社A, B, C...）
       └─ Kiosk User（キオスク利用者 - 認証不要）
```

| ロール | 権限 |
|--------|------|
| admin | オペレーター管理、全データ閲覧、システム設定 |
| operator | 自社物件取り込み・編集、自社顧客管理、営業メール配信 |
| kiosk_user | AI会話、物件検索（会話内）、内見予約（会話内）※認証不要 |

**注意**: 顧客（kiosk_user）は店舗設置キオスク端末から利用し、ログイン不要。会話完了後に連絡先を入力し、その後は実際の従業員が対応。
